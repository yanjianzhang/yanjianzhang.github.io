2021-11-26

数学表达式渲染

试过了很多方法都不行，最后升级了hexo，使用了一下目录
https://github.com/hexojs/hexo-math
还是要保留原有的 hexo-renderer-marked

npm install hexo-renderer-marked --save

表格渲染

表格前后需要空一到两行，才能在部署到网站上成功渲染

toc目录

每个目录的层级必须连续，不能从层级1跳到层级3，否则toc目录渲染失败

图片

图片的路径很难统一，最后笔者是修改了_config.yml, 使得路由比较简单，使用相对路径

permalink: :title/

hexo 常用命令

hexo clean  # 清空public目录
hexo g      # 生成public文件
hexo s      # 本地服务测试
hexo d      # 部署到远端
