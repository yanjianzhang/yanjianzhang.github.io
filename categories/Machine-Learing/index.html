机器学习评价指标

2018-11-09

分类问题的评价指标 * Relevant，正类 Nonrelevent，负类 Retrieved 被检索到 TP FP Notretrieved未检索到 FN TN 准确率是：对于给定的测试数据集，分类器正确分类的样本数与总样本数之比 A(accurate) = \frac{TP+TN}{TP+TN+FP+FN}精确率是：它计算的是所有被检索到的item中,”应该被检索到”的item占的比例。 P(percision) = \frac{TP}{TP+FP}召回率是：它计算的是所有检索到的item占所有”应该检索到的item”的比例。 R(recall) = \frac{TP}{TP+FN}综合评价指标F-measure是精确度和召回率的综合 F = \frac{(a^2+1)PR}..
